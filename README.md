# Tip-Calculator

A responsive tip calculator and bill splitter built using HTML, CSS, and JavaScript.

## Features

- Live calculations
- Tip preset buttons
- Custom tip input
- Inline validation
- Responsive design
- Reset functionality
- Accessible keyboard-friendly inputs

---

## How to Run

1. Download or clone the project
2. Open the project folder
3. Open `index.html` in your browser

No installation required.

---

## Deployment

You can deploy easily using:
- Netlify
- Vercel
- GitHub Pages

- HTML
